Customization Guide

How to Change Lobby Colors

Learn how to edit your lobby appearance by changing host colors, icons, names, and tag gradients inside your language file.

Before You Start

When you first download the mod, you will usually see a template.dat file in your language folder. To use it, rename template.dat to english.dat, then open it and fill in the fields you want to customize.

Steam\steamapps\common\Among Us\Language
Always make a backup before editing your file. If something breaks, restore the backup and try again. Most mods will do this automatically.
Step 1: Go to your Among Us (Mod)\Language folder.
Step 2: Find template.dat.
Step 3: Rename it to english.dat.
Step 4: Open the file in a text editor and fill in your values.
Step 5: Save the file and relaunch the game if needed.

What You Can Customize

Host name color Host icon Icon color Displayed host text VIP tags Moderator tags Admin tags Much more

Host text

HostText: Changes the styled word shown for the host.

Host text color

HostColor: Changes the main host color.

Icon

Icon: Changes the symbol shown next to the host.

Icon color

IconColor: Changes the icon color.

Name color

NameColor: Changes the displayed host/player name color.

Main Fields Explained

Displayed host label

HostText:

This is where you can put full styled text using color tags. You can make each letter a different color to create a gradient.

HostText:<color=#570202>♡</color><color=#7A0203>H</color><color=#9E0305>o</color><color=#C10307>s</color><color=#FC0509>t</color>
Flat color values

HostColor: / IconColor: / NameColor:

These use normal hex color codes. Example: #570202 or #FC0509.

HostColor:#570202 IconColor:#FC0509 NameColor:#9E0305
Host icon symbol

Icon:

This changes the symbol that appears by the host name. You can use things like ♡, ♥, ✿, or another supported symbol.

Icon:♡

Example: Red Host Setup

This is a clean example using the deep red to bright red gradient style.

Tag Colors

Tags like VIP, Moderator, and Admin can also use gradients. You can color each character separately the same way you do with host text.

Admin tag example

AdminTag:<font="Barlow-Light SDF"><color=#570202>♡</color><color=#7A0203>C</color><color=#9E0305>o</color><color=#C10307>-</color><color=#FC0509>H</color><color=#C10307>o</color><color=#9E0305>s</color><color=#7A0203>t</color><color=#570202>♡</color></font>

VIP tag example

Moderator tag example

ModeratorTag:<font="Barlow-Light SDF"><color=#FF0F7B>♡</color><color=#F83E64>G</color><color=#F5694C>a</color><color=#F6803B>t</color><color=#F89B29>o</color><color=#FF0F7B>s</color><color=#F83E64>♡</color></font>

Quick Tips

  • Use hex colors like #570202 or #FF0F7B.
  • Wrap each letter in its own <color=...> tag for gradients.
  • Keep a backup of both template.dat and english.dat.
  • If the game does not update right away, save the file and restart the game.
  • Copy working examples first, then change colors one step at a time.

This guide is for editing your language file and customizing how your lobby text and tags appear in-game.
