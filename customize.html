Customization Guide

How to Customize Lobby Colors

Learn how to edit your lobby appearance by changing host text, host colors, icons, player name colors, and tag styling inside your language file. This version starts with simple solid-color examples first, then moves into gradients later for advanced users.

Before You Start

When you first download the mod, you will usually see a template.dat file in your language folder. To use it, rename template.dat to english.dat, then open it and fill in the values you want to customize.

Steam\steamapps\common\Among Us\Language
Always make a backup before editing your file. If something breaks, restore the backup and try again.

Beginner

Solid Colors First

If you are new to this, start here. Solid colors are the easiest to test because each line only uses one hex color code. Once this works, you can move on to gradients later.

Flat color values

HostColor: / IconColor: / NameColor:

These use normal hex color codes. Examples: #3A86FF, #8338EC, or #06D6A0.

HostColor:#3A86FF IconColor:#8338EC NameColor:#06D6A0
image/example.png
Simple host text

HostText:

This colors the whole word using one single color.

HostText:<color=#3A86FF>Host</color>
image/example.png
Host icon symbol

Icon:

This changes the symbol that appears next to the host name. You can try symbols like ★, ✿, ◆, or ♣.

Icon:★

Main Fields Explained

Displayed host label

HostText:

This is the text that gets shown for the host label. It can be plain, one solid color, or fully styled with multiple color tags.

HostText:<color=#3A86FF>Host</color>

Advanced

Gradient Text and Tags

Gradients are optional and a little harder to edit. Instead of using one color, you wrap each character in its own <color=...> tag. This lets you create fading or rainbow-style text.

Do this only after your basic solid-color setup is already working.

Gradient host text

Here each letter is given a slightly different color.

HostText:<color=#3A86FF>H</color><color=#5F9CFF>o</color><color=#83B1FF>s</color><color=#A7C7FF>t</color>
image/example.png

VIP tag example

Tags like VIP can also use gradients the same way host text does.

VIPTag:<font="Barlow-Light SDF"><color=#8338EC>V</color><color=#9B5DE5>I</color><color=#C77DFF>P</color></font>
image/example.png

Moderator tag example

You can also build softer gradients using greens, blues, purples, or any palette you want.

ModeratorTag:<font="Barlow-Light SDF"><color=#06D6A0>M</color><color=#38E0B8>o</color><color=#72EFDD>d</color><color=#A8F5E5>s</color></font>
image/example.png

Admin tag example

Longer words or wrapped labels work the same way. Just keep adding color tags around each character or symbol.

AdminTag:<font="Barlow-Light SDF"><color=#4D96FF>C</color><color=#6AA8FF>o</color><color=#8AB9FF>-</color><color=#A9CBFF>H</color><color=#8AB9FF>o</color><color=#6AA8FF>s</color><color=#4D96FF>t</color></font>

Good Workflow

Best way to edit

  1. Start with one solid color.
  2. Test that the file works.
  3. Change your icon.
  4. Change your host text.
  5. Only then try gradients.
image/example.png

What to do if it breaks

  • Restore your backup.
  • Check for missing </color> tags.
  • Make sure every hex code starts with #.
  • Test one line at a time.
  • Restart the game after saving.
image/example.png

Quick Tips

  • Use hex colors like #3A86FF, #8338EC, or #06D6A0.
  • Solid colors are easier to debug than gradients.
  • Wrap each letter in its own <color=...> tag for gradients.
  • Keep a backup of both template.dat and english.dat.
  • If the game does not update right away, save the file and restart the game.
  • Copy working examples first, then change colors one step at a time.
