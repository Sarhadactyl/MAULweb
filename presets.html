Using Preset Files

  1. Rename the downloaded preset to Options.json.
  2. Place it in the mod’s SaveData folder, e.g. ...\Among Us\<mod-name>-DATA\SaveData.
  3. Open the game, choose the preset in the mod’s settings, and play.

If you can’t see file extensions or the *-DATA\SaveData folder, click "View" then "Show" then “Hidden items” in the file explorer settings.
Please refer to the lotus presets tab for instructions on how to add lotus presets.
