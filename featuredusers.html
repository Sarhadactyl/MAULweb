Featured Users

Featured Users showcase the people who help make MAUL special — hosts, developers, contributors, helpers, staff, creators, and more. If you think you qualify, check the requirements below and apply through the correct route.

Home Join MAUL

The Featured Users section highlights trusted, active, and notable members of the MAUL community. This includes Verified Hosts, returning hosts, community helpers, server staff, mod developers, major contributors, content creators, and users with standout activity, loyalty, or consistency.

If you believe you qualify, please follow the application instructions below and contact Sarha if needed.

Verified Host Perks

Being a Verified Host comes with extra recognition and useful server permissions.

Lobby Posting Access

Can post and ping in verified-host-lobbies as the host.

Event Hosting

Can hold events in the Event section/category.

Custom Role

Can create a custom role using @Booster Bot.

VC Assignment

Can assign @Blacklisted VC.

Exclusive Access

Access to the Verified Host category and exclusive-chat.

Server Recognition

Displayed as an officially recognized MAUL host with a dedicated featured badge/status.

Featured User Badges

These are the badge categories currently used for Featured Users. Badge appearance may vary, and some are tied to specific recognition, activity, region, or hosting milestones.

Tester / Bug Hunter

For users who test features, report bugs, or help catch issues affecting mods, systems, or community tools.

Developer

For mod developers or known community bot developers recognized within the modded Among Us space.

Contributor

For users who contribute to a mod or a known community bot through code, ideas, testing, assets, or support.

Newest VH

Highlights the newest member added to the Verified Host lineup.

Verified Host

Main Verified Host recognition badge. This may later be replaced or expanded with numbered badges.

Lobby Milestones

Flame badges mark amount of lobbies hosted, including 50, 75, 100, 150, 200, 300, 400, and 500+ lobbies.

Helper

For members active in support or welcome-chat and known for helping others.

Staff

For official MAUL server staff members.

Booster

For members who have boosted the server for 3+ months.

Large / Trusted Server Owner

For owners of large or trusted servers recognized by the community.

Large / Trusted Server Admin

For admins of large or trusted servers recognized by the community.

Returning / OG Host

For hosts who were active prior to 2024 and returned, or long-time OG hosts.

Hosts EHR the Most

Recognition for the user most associated with hosting Endless Host Roles.

Hosts Project Lotus the Most

Recognition for the user most associated with hosting Project Lotus.

Hosts TOUM the Most

Recognition for the user most associated with hosting TOUM.

Hosts MGM the Most

Recognition for the user most associated with hosting MGM.

Most Active Chatter

For standout chat activity and consistent community engagement.

Content Creator / Contributor

For recognized creators or contributors making community content, media, guides, or promotional material.

Most Consistent Host

For hosts who show strong long-term consistency and reliability in their lobby schedule.

Event Host

For users who have hosted an event in MAUL.

Veteran User

Recognizes duration of time in the server or being a Verified Host for x amount of years.

Region Badges

Region-based recognition for North America, Europe, and Asia.

How to Apply

Developers, staff, Level 175+ and Verified Hosts can utilize this channel. Users who are meet these requirements can DM Sarha or open a ticket.
